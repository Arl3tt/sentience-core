## What's New

### Brain Wave Classifier
- Frequency band analysis (Delta, Theta, Alpha, Beta, Gamma)
- Cognitive state detection (alert, focused, relaxed, drowsy, stressed)
- Multi-channel support with consensus voting

### Motor Imagery CNN
- Deep learning classification for 4 motor imagery types
- Heuristic fallback when PyTorch unavailable
- Feature extraction and model training support

### P300 Speller
- Event-related potential detection
- 6×6 character speller matrix interface
- SNR calculation and multi-trial accumulation

### Neurofeedback Loop
- Real-time feedback generation
- Multiple modalities (visual, auditory, proprioceptive)
- 4 feedback targets (relaxation, focus, arousal, calmness)
- Session tracking and reporting

### Hybrid BCI
- Multi-paradigm fusion (motor imagery, P300, SSVEP)
- Adaptive weighting based on paradigm performance
- 3 fusion methods (voting, weighted, averaged)
- Performance metrics tracking

### ASD Attention Analysis
- ASD-specific attention pattern detection
- Social attention analysis
- Repetitive pattern detection
- Executive function assessment
- Longitudinal tracking and trending

## Quick Start

### 1. Import Tools
```python
from core.tools import (
    classify_brainwave_bands,
    classify_motor_imagery,
    create_hybrid_bci,
    create_neurofeedback_session,
    create_asd_analyzer
)
```

### 2. Use in Your Agent
```python
# Analyze EEG
eeg_data = get_eeg_data()  # Your EEG acquisition

# Brain state
state = classify_brainwave_bands(eeg_data)

# Motor command
imagery = classify_motor_imagery(eeg_data)

# Multi-paradigm BCI
bci = create_hybrid_bci()
command = bci.process_multi_paradigm(eeg_data)

# Real-time feedback
nf = create_neurofeedback_session(target="focus")
feedback = nf.process_eeg_chunk(eeg_data)

# Clinical analysis
asd = create_asd_analyzer()
profile = asd.analyze_attention_profile(eeg_data)
```
